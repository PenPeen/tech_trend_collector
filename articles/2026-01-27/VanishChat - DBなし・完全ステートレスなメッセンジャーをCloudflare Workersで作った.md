## 要約

ご提示いただいた記事「VanishChat：DBを一切使わない究極の匿名メッセンジャーを作ってみた」の内容を要約しました。

### 1. 概要
「VanishChat」は、プライバシーと匿名性を最大限に高めることを目的とした、**データベース（DB）を一切使用しない**リアルタイム・チャットアプリケーションです。メッセージがサーバーやストレージに保存されないため、物理的にログが残らない仕組みになっています。

### 2. 主な特徴
*   **完全な匿名性:** ユーザー登録やログインが不要で、ニックネームだけで利用可能です。
*   **データ非保存:** メッセージはDBに保存されず、メモリ上でのみ処理されます。
*   **ルーム制:** 任意のルーム名を作成し、そのURLを共有するだけで特定の相手とチャットが可能です。
*   **自動消滅:** ブラウザを閉じたり、サーバーが再起動したりするとメッセージ履歴は完全に消失します。

### 3. 技術スタック
*   **フロントエンド:** Next.js (React), Tailwind CSS
*   **バックエンド:** Node.js (Express)
*   **リアルタイム通信:** Socket.io
*   **インフラ:** Render（無料枠を利用したデプロイ）

### 4. 仕組みのポイント
*   **WebSocketの活用:** メッセージのやり取りにはSocket.ioを使用。送信されたメッセージは、その時同じ「ルーム」に接続しているクライアントに即座に転送（ブロードキャスト）されるだけです。
*   **一時的な保持:** 新しく入室したユーザーが直近のやり
